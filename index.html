SQL QUERYING

This project demonstrates my understanding of basic SQL commands used for data exploration using the Microsoft SQL server Interface. In this project, I have used Northwind dataset available online. This dataset is explored using SQL Data Definition Language(DDL) such as create, drop table and alter table, Data Manupulation language(DML) such as insert, update and delete, Data Querry Language(DQL) such as select statement, where clause, application of AND/OR operators, looking into NULL/NOT NULL values, use of between operator, wildcards and aliases, date calculations, aggregate functions for summerising data, string functions, order by, group by and having clauses, RANK() function, mathematical expressions, simple pivot tables, inner join, union and sub queries.
